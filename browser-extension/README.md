# Elysium Chrome Extension

Chrome extension для поддержки `ely://` протокола.

## Установка

1. Открой Chrome и перейди в `chrome://extensions/`
2. Включи "Режим разработчика" (Developer mode)
3. Нажми "Загрузить распакованное расширение" (Load unpacked)
4. Выбери папку `browser-extension/`

## Использование

- Клики по `ely://` ссылкам автоматически открывают контент через Web Gateway
- Навигация на `ely://` URLs перехватывается и перенаправляется
- Адресная строка показывает оригинальный `ely://` URL (насколько возможно)

## Настройки

Нажми на иконку расширения чтобы настроить порт Web Gateway (по умолчанию 17081).

## Примечания

- Chrome не позволяет полностью изменить протокол в адресной строке из соображений безопасности
- Расширение перехватывает `ely://` и перенаправляет на Web Gateway
- Title страницы обновляется на `ely://` URL
- История браузера хранит оригинальный `ely://` URL в state
